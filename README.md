# Tailwind HTML Starter - Quick Start

Minimal HTML + Tailwind CSS project with PostCSS.

---

## Quick Steps

1. **Clone the repository**

```bash
git clone <your-repo-url>
cd html-tailwind-starter
```

2. **Install dependencies**

```bash
pnpm install
```

> If pnpm is not installed globally:

```bash
npm install -g pnpm
```

3. **Start development**

```bash
pnpm run dev
```

* Watches CSS and rebuilds automatically.

4. **Open in browser**

* Open `index.html` in the project root.

5. **Edit files**

* Edit `index.html` or `assets/css/input.css`. CSS updates automatically.

---

You're ready to build your Tailwind-powered HTML project!
